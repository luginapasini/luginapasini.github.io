About


Halo nama saya Lugi Napasini, saya adalah freelancer pada bidang pekerjaan Web Programing, Web Design, Logo Design, Sound Design, Teknisi jaringan, Akuntansi, dan Administrasi. Selain bekerja sebagai freelance saya membuka usaha voucher iuran internet, dari sinilah saya menjadi teknisi jaringan dari mulai merancang topologi, setting perlatan dan instalasi jaringan. Dari sini saya dapat membuat konfigurasi dengan Router OS tentunya dengan Routerboard Mikrotik untuk membuat jaringan LAN.

Saya pernah menjadi seorang karyawan dari perusahaan swasta yang bergerak di bidang jasa keuangan. Di perusahaan tersebut saya bekerja sebagai staff operational dengan fungsi administrasi. Saya merupakan lulusan dari Sekolah Tinggi Ilmu Ekonomi Sebelas April Sumedang pada program Starata 1, program studi Manajemen. Sebelumnya saya merupakan lulusan dari SMK Negeri 1 Sumedang Jurusaan Teknik Komputer Jaringan.
